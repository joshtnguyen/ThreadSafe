# CMPE-131-Term-Project
## Name of Application:
Secure E2EE Web Messaging App  

### Overview:
A secure, scalable messaging platform implementing true end-to-end encryption (E2EE) for both 1-to-1 and group communications. Built with Flask backend and modern web frontend, this application ensures message privacy through cryptographic protocols including Elliptic Curve Cryptography (ECC), AES-256 symmetric encryption, and the Double Ratchet algorithm for perfect forward secrecy.
